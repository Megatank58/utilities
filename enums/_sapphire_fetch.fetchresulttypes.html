Buffer

Buffer: = "buffer"
  • Defined in packages/fetch/src/lib/types.ts:15

Returns only the body, as a Buffer. Similar to Body.blob().

JSON

JSON: = "json"
  • Defined in packages/fetch/src/lib/types.ts:11

Returns only the body, as JSON. Similar to Body.json().

You should provide your own type cast (either through the generic return type, or with as <type>) to the response to define the JSON structure, otherwise the result will be unknown.

Result

Result: = "result"
  • Defined in packages/fetch/src/lib/types.ts:23

Returns the entire response and doesn't parse the body in any way.

Text
