Module @sapphire/ts-config

@sapphire/ts-config

TypeScript configuration for all Sapphire Community repositories.

Installation

First install the Peer Dependencies

yarn add -D @sapphire/ts-config

Usage

You can use @sapphire/ts-config's tsconfig.json by extending it in yours:

{
    "extends": "@sapphire/ts-config"
}

All of its properties are defaulted for bleeding-edge TypeScript options, you may extend this to include your own configuration options as well.
